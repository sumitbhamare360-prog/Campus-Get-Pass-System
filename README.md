# Campus Digital Gate Pass & Visitor Management System

A web-based system designed to manage campus visitors and generate digital gate passes efficiently.
The system replaces traditional paper-based visitor logs with a secure digital platform that tracks visitor entry, approval, and exit in real time.

---

## 📌 Project Overview

Traditional visitor management in campuses is usually done through manual registers, which leads to errors, delays, and security risks.

This project introduces a **Digital Gate Pass System** that allows visitors to register online, get approval from the host, and generate a digital gate pass for entry into the campus.

The system improves **security, efficiency, and record keeping** by automating entry tracking and approval workflows.

---

## 🎯 Objectives

* Replace manual gate pass systems with a digital platform
* Improve campus security and visitor tracking

## 🏗 System Architecture

```
Visitor → Register Visit
      ↓
Host → Approves Request
      ↓
System → Generates Digital Gate Pass
      ↓
Security → Verifies Entry
      ↓
System → Logs Entry & Exit
```

## ⚙ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/sumitbhamare360-prog/Campus-Get-Pass-System.git
```

### 2. Go to the project directory

```bash
cd Campus-Get-Pass-System
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

### 5. Open in browser

```
http://localhost:5173
```
